# 🚇 Uber Metro Integration Case Study

## Overview

A Product Management case study exploring how Uber can improve first-mile and last-mile connectivity for Pune Metro commuters through an integrated Metro + Uber Auto booking experience.

## Problem

Metro users face friction when planning end-to-end journeys, requiring multiple apps and separate bookings for metro tickets and last-mile transportation.

## Proposed Solution

Design an integrated **Metro + Uber Auto Bundle** that enables commuters to plan, book, and pay for their complete journey within Uber.
